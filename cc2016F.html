Creative Coding F 2016

Creative Coding employs computer programming as a means of creative expression. “Algorithmic” and generative art, interactive web apps, data visualizations, and virtual reality are just a few examples of the creative potential of code explored through projects. Students will develop foundational computational thinking skills in the areas of abstraction, boolean logic, algorithms, complexity, and data analysis. Projects include:

  1. Interactivity: In Project 1 students demonstrate proficiency with JavaScript fundamentals and the p5.js library by developing interactive, single-page websites.
  2. Data Visualization: In this project students create data visualizations on the web using personally-relevant data.
  3. Virtual Reality: In the third project students develop web virtual reality apps that can be viewed using Google Cardboard. This project relies on three.js, a JavaScript library designed for 3D graphics.
